# Mac MD Website

This is the marketing website for Mac MD, hosted on GitHub Pages.

## Setup GitHub Pages

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select `main` (or your default branch) and `/docs` folder
5. Click "Save"

Your site will be available at: `https://brainvat.github.io/markdown-editor/website/`

## Local Development

Simply open `index.html` in your browser to preview locally.

## Structure

- `index.html` - Main landing page
- `styles.css` - All styling (responsive, dark mode support)
- `README.md` - This file

The site uses the `mac_md.png` logo from the repository root.
